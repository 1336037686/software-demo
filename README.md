# 目录结构



|-wg-client-simple，客户端主体

|-wg-server-simple，更新服务器，用于保存新版本软件

|-wg-update-simple，更新程序，用于更新客户端

|-example 项目案例，打包后的jar包命名格式必须和里面的相同



客户端jar包命名：wg-client-simple.jar

更新程序jar包命名：wg-update-simple.jar
